# Implementation-of-4th-Order-Chebyshev-Bandpass-Filter
## 📌 Project Overview
Design and simulation of a **4th order Chebyshev bandpass filter** for **UHF applications**.  
The project demonstrates practical RF filter design using microstrip resonators and validates performance through simulation, achieving low insertion loss and high return loss.

**Key Specifications:**
- Center Frequency: 800 MHz  
- Fractional Bandwidth: 6%  
- Passband Ripple: 0.043 dB  
- Return Loss: ≤ –20 dB  
- Characteristic Impedance: 50 Ω  
- Tool Used: AutoCAD 360  
- Application: UHF RF front-end filtering  

---

## 📈 Project Summary
This project implements a **Chebyshev bandpass filter**, which provides a sharper roll-off than Butterworth filters due to controlled passband ripple.  

The design workflow includes:  
1. **Theoretical Design:** Chebyshev polynomial coefficient calculation  
2. **Circuit Implementation:** Microstrip layout  
3. **Simulation:** Frequency response, insertion loss, and return loss validation  

The project demonstrates the translation of **theory into a practical RF filter**, with attention to both performance and real-world implementation.
